# Alzheimer's Disease Drug Repurposing Project

This project uses the DRKG (Drug Repurposing Knowledge Graph) and pre-trained TransE embeddings to predict potential drug candidates for treating Alzheimer's disease.

## Project Structure

```
alzheimers_drug_repurposing/
├── alzheimers_drug_repurposing.py    # Main analysis script
├── utils.py                           # Download utility for DRKG data
├── alzheimers_known_drugs.tsv        # Known AD drugs for validation
├── neo4j_import_predictions.cypher   # Neo4j visualization queries
├── README.md                          # This file
├── drkg/                              # Created by utils.py
│   ├── drkg.tsv                      # Full knowledge graph
│   └── embed/                         # Pre-trained embeddings
│       ├── entities.tsv
│       ├── relations.tsv
│       ├── DRKG_TransE_l2_entity.npy
│       └── DRKG_TransE_l2_relation.npy
├── infer_drug.tsv                    # Generated by script
└── alzheimers_top100_predictions.tsv # Output predictions
```

## Usage

### Step 1: Run the Analysis

```bash
python alzheimers_drug_repurposing.py
```

**What happens:**
1. Downloads DRKG data (first run only, ~5-10 minutes)
2. Extracts DrugBank compounds from DRKG
3. Loads pre-trained TransE embeddings
4. Calculates prediction scores for all drugs
5. Ranks drugs by treatment potential
6. Outputs top 100 predictions to console and file

**Output:**
- Console: Top 100 ranked drugs with scores
- File: `alzheimers_top100_predictions.tsv`

### Step 2: (Optional) Visualize in Neo4j

If you want to explore the knowledge graph visually:

1. **Start Neo4j and create a new database**

2. **Add headers to DRKG file:**
   ```bash
   # On Mac/Linux:
   echo -e "source\trelation\ttarget" | cat - drkg/drkg.tsv > drkg/drkg_with_headers.tsv
   
   # On Windows (PowerShell):
   "source`trelation`ttarget" | Out-File -Encoding utf8 drkg_with_headers.tsv
   Get-Content drkg/drkg.tsv | Add-Content drkg_with_headers.tsv
   ```

3. **Copy files to Neo4j import folder:**
   ```bash
   # Find your Neo4j import folder location
   # Usually: ~/.Neo4jDesktop/relate-data/dbmss/dbms-xxxxx/import/
   
   cp drkg/drkg_with_headers.tsv /path/to/neo4j/import/
   cp alzheimers_top100_predictions.tsv /path/to/neo4j/import/
   ```

4. **Run the import queries from `neo4j_import_predictions.cypher`**

## Understanding the Output

### Prediction Scores

The output file contains:
- `rank`: Position (1-100, lower is better)
- `drug_id`: Full DRKG identifier (e.g., Compound::DB00843)
- `drug_name`: DrugBank ID (e.g., DB00843)
- `score`: Prediction score (closer to 0 = stronger prediction)

**Score Interpretation:**
- `-0.09` = Excellent prediction (very strong)
- `-0.15` = Good prediction (strong)
- `-0.30` = Moderate prediction
- `-1.00+` = Weak prediction

Scores are **always negative** due to the logsigmoid function. Values closer to 0 indicate higher confidence.

### Top Predictions

The top predictions typically include:
1. **Known AD drugs** (validation of the method):
   - Galantamine (DB00674)
   - Rivastigmine (DB00989)
   - Memantine (DB01043)

2. **Novel candidates** for further investigation

## Methodology

### Data Source
- **DRKG**: 97,238 entities, 5.8M relationships from 6 databases
- **Sources**: DrugBank, Hetionet, GNBR, String, IntAct, DGIdb

### Algorithm
- **TransE embeddings**: Vector representations of entities and relations
- **Scoring function**: d = γ - ||h + r - t||₂, score = logsigmoid(d)
- **Relations used**: Treatment-specific (Hetionet::CtD, GNBR::T)

### Validation
- Top predictions include FDA-approved AD medications
- Network analysis reveals biological mechanisms

## Troubleshooting

### Download Issues

If automatic download fails:
1. Manually download from: https://github.com/gnn4dr/DRKG/releases
2. Extract `drkg.tar.gz` to create a `drkg/` folder
3. Run the script again

### Memory Issues

If you run out of RAM:
- Close other applications
- The embeddings require ~2-3 GB of RAM
- Use a machine with at least 8 GB RAM

### Import Errors

## Neo4j Visualization

Once data is imported, try these queries:

**View top 10 predictions:**
```cypher
MATCH (drug:PredictedDrug)
WHERE drug.rank <= 10
RETURN drug
ORDER BY drug.rank;
```

**Explore mechanisms:**
```cypher
MATCH path = (drug:PredictedDrug)-[:SAME_AS]->(:Entity)
             -[:DRKG_REL*1..2]-
             (disease:Entity {drkg_id: 'Disease::MESH:D000544'})
WHERE drug.rank <= 10
RETURN path
LIMIT 200;
```
